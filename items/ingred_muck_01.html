Muck

Muck is a common ingredient found in muckspunges, which grow mostly in coastal regions. Muck is commonly harvested using a Muck Shovel, although it is not required to acquire one, nor does it help in any way to carry one.

Type Ingredient
Cost 1
Weight 0.1
Code ingred_muck_01

Effects

  • Detect Key
